# Multi-Location

## Purpose
Multi-restaurant management dashboard — compare performance across locations, manage menus and settings centrally.

## Route
`/multi-location`

## Components
- **MultiLocationDashboard** (`os-multi-location-dashboard`) — Location comparison grid, cross-location sales, menu sync, settings propagation

## Services
- `MultiLocationService` — Location list, cross-location analytics, menu sync

## Models
- `multi-location.model` — Location, LocationComparison, SyncStatus
